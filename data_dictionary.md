# Data Dictionary: merged_school_crime.csv

| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| **School_ID** | Integer | Unique identifier assigned to each Chicago Public School. |
| **Name_of_School** | String | The official name of the school. |
| **Elementary, Middle, or High School** | String | ES: elementary; MS: middle; HS: high school. |
| **Safety_Score** | Float | Student Safety score (1-99) from 5 Essentials survey. |
| **Average_Student_Attendance** | String | Average daily student attendance %. |
| **ISAT_Exceeding_Reading_Percent** | Float | Percentage of students exceeding reading standards. |
| **Community_Area_Name** | String | Chicago community area name where the school is located. |
| **Community_Area_Number** | Integer | The official index number for the neighborhood. |
| **Latitude** | Float | Geographic latitude coordinate of the school. |
| **Longitude** | Float | Geographic longitude coordinate of the school. |
| **Ward** | Integer | City Council district number used to join datasets. |
| **Crime_Count** | Integer | Total crimes reported in the Ward (2011-2012). |
